# PickEat AI Coach 🥗

## Introduction

PickEat AI Coach is an AI-powered web application that helps users build healthier eating habits by analyzing nutrition labels and ingredient lists. Users can simply take a photo of a product's nutrition facts and ingredients, and the AI analyzes the contents to provide easy-to-understand health information.

## Features

- AI-powered analysis of nutrition labels and ingredient lists
- Displays key nutrients such as carbohydrates, protein, and fat
- Identifies sugar substitutes and food additives
- Highlights ingredients that may require caution
- Tracks and manages users' nutritional intake

## Getting Started

1. Clone or download this repository.
2. Install the required dependencies.
3. Configure the `.env` file.
4. Run `npm install`.
5. Start the development server with `npm run dev`.
